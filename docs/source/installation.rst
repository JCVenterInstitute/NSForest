Installation
============

To install NSForest via Github: 

.. code-block:: console

   $ git clone https://github.com/JCVenterInstitute/NSForest.git
   $ cd NSForest
   $ conda env create -f nsforest.yml
   $ conda activate nsforest
   $ cd tutorials

To install NSForest with pip: 

.. code-block:: console

   $ pip install nsforest
